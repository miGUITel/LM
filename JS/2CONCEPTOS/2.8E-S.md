# **Entrada y salida de datos en JavaScript: `prompt()` y `console.log()`**

Hasta ahora tus programas han mostrado mensajes en la consola utilizando `console.log()`. Pero un programa es mucho más útil cuando **interactúa con el usuario**, igual que una aplicación en el móvil te pregunta por tu nombre, tu edad, o una opción a elegir.

En esta introducción aprenderás a **solicitar datos** de forma sencilla utilizando `prompt()`, una función integrada en el navegador que permite recoger una entrada escrita por el usuario.

---

## **1. `console.log()` → salida de información**

Es la forma más básica de mostrar resultados en JavaScript.
Envía mensajes a la **consola del navegador**, que se abre con **F12 → Consola** en Firefox.

### Ejemplo:

```javascript
console.log("Bienvenido al programa");
```

Lo usaremos para mostrar respuestas, resultados y mensajes informativos.

---

## **2. `prompt()` → entrada de información**

La función `prompt()` muestra una pequeña ventana emergente donde el usuario puede escribir un valor.
JavaScript recoge ese valor y permite guardarlo en una variable.

### Ejemplo básico:

```javascript
let nombre = prompt("¿Cómo te llamas?");
console.log("Hola, " + nombre);
```

Cuando el usuario introduce su nombre, el programa lo almacena en la variable `nombre` y luego lo muestra en la consola.

---

## **3. Importante: `prompt()` siempre devuelve texto**

Incluso si el usuario escribe un número, JavaScript **lo tratará inicialmente como una cadena**.

```javascript
let edad = prompt("Introduce tu edad");
console.log(typeof edad);  // Resultado: "string"
```

### Para convertir ese texto en número:

```javascript
let edad = Number(prompt("Introduce tu edad"));
console.log(edad + 1);  // Ahora sí suma correctamente
```

---

## **4. Ejemplos típicos que tus alumnos pueden probar**

### **4.1 Clasificar una edad**

```javascript
let edad = Number(prompt("¿Qué edad tienes?"));

if (edad >= 18) {
    console.log("Eres mayor de edad");
} else {
    console.log("Eres menor de edad");
}
```

---

### **4.2 Pedir dos números y mostrar su suma**

```javascript
let a = Number(prompt("Primer número:"));
let b = Number(prompt("Segundo número:"));

console.log("La suma es:", a + b);
```

---

### **4.3 Repetir un bucle en función de la entrada del usuario**

```javascript
let veces = Number(prompt("¿Cuántas veces quieres repetir el mensaje?"));

for (let i = 1; i <= veces; i++) {
    console.log("Repetición", i);
}
```

---

### **4.4 Pedir datos dentro de una función**

```javascript
function saludar() {
    let nombre = prompt("Nombre del usuario:");
    console.log("Encantado, " + nombre);
}

saludar();
```

---

## **5. Ventajas y limitaciones de `prompt()`**

### ✔️ Ventajas

* Es muy fácil de usar.
* No necesita HTML adicional ni manipular el DOM.
* Ideal para aprender la lógica básica de entrada-proceso-salida.

### ❌ Limitaciones

* Es poco estética.
* Bloquea la ejecución hasta que el usuario responde.
* No se usa en aplicaciones web modernas.
