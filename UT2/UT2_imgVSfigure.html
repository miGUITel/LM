Comparación entre img y figure

Ejemplo usando img

En este ejemplo, se utiliza la etiqueta img para insertar una imagen directamente en el documento. La imagen se muestra sin ninguna leyenda o texto asociado. Este es el uso más común cuando solo necesitas mostrar una imagen sin necesidad de proporcionar una explicación adicional.

La etiqueta img es simple y adecuada para mostrar imágenes que no requieren contexto adicional o explicación.

Ejemplo usando figure

En este ejemplo, la imagen se coloca dentro de una etiqueta figure, que es un contenedor semántico. Además, se agrega una leyenda utilizando figcaption para proporcionar un contexto o descripción de la imagen.

El uso de figure junto con figcaption es ideal cuando la imagen necesita ser interpretada o explicada en contexto, lo que mejora la accesibilidad y la comprensión del contenido visual.

Ejemplos de img con width y height

A continuación, se muestra la misma imagen utilizando diferentes combinaciones de los atributos width y height:

1. Solo width (ancho) establecido

En este ejemplo, solo se establece el atributo width a 40. La altura se ajusta proporcionalmente.

2. Solo height (altura) establecido

Aquí, solo se establece el atributo height a 200. El ancho se ajusta proporcionalmente.

3. Width y height establecidos (no proporcionales)

Aquí, se establecen ambos atributos width="50" y height="300" con valores que no mantienen las proporciones originales de la imagen, lo que provoca una distorsión.
